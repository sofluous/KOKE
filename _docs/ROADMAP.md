# ROADMAP

## Stage 1 (current)
- Modular Three.js app scaffold.
- Vertex-cell growth simulation with environmental controls.
- Hybrid moss rendering (shader coverage + instanced clumps).

## Stage 2
- GLTF mesh import pipeline with preprocessing for simulation cells.
- Multiple species and competitive colonization rules.
- Parameter presets for biome scenarios.

## Stage 3
- Export/import of simulation states.
- Advanced shading (wetness, subsurface hints, species color variation).
